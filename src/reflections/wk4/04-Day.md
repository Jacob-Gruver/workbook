## What does REST stand for, and in simple terms what does it mean?
 REST represents REpresentational State Transfer, which means making transfers within code. It means the user can add information to the resource or remove it/edit it.

## What does Stateless mean?
 Stateless is basically a state of the server. It is when t doesnt remember if the user accessed or did anything with the data within the resource.

## What url pattern is used when writing a RESTful api?
 A RESTful api pattern usually include two sections. One is the username to identify the user, and the HTTP Get.
